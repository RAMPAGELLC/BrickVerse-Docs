# Types

The **Types** section contains every class, object, service, and data structure available to BrickVerse scripts.

Each page documents a single type and includes its inheritance hierarchy, properties, methods, events, and usage notes. Types form the foundation of the BrickVerse API and represent everything from world objects and players to mathematical structures and engine services.

## Contents

Type pages include:

- Class overview
- Inheritance
- Properties
- Methods
- Events
- Server/client availability
- Return values
- Parameter types
- Related API types
