# Enums

The **Enums** section contains every enumeration used throughout the BrickVerse engine.

Enums define a fixed set of named values that are used by properties, methods, and events. Using enum values instead of raw numbers or strings makes scripts easier to read and less prone to errors.

## Contents

Each enum page contains:

- All available values
- Descriptions for each value
- Related API references

Examples:

```lua
part.Material = Material.Metal

camera.Mode = CameraMode.Scriptable
```
